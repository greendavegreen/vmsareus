Change password on first login

We STRONGLY recommend that you change your password upon first connection. Right-Click on Cygwin-terminal and run it as administrator. Then use this command with your new password substituted: 

net user {{ request.user.username }} YOUR_NEW_PASSWORD
 

Create a symlink to your codebase

A file titled linkDevToE has been placed on your desktop. If you right click on it and choose 'Run as administrator', Windows will create a link from your cygwinHome/dev to the directory holding your workspace. This is a one-time task and from thenceforth you will be able to start a cygwin terminal and cd dev/star to get to the root of your git repo as you may be familiar with in your local workstation.
